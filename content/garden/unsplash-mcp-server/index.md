[Unsplash MCP Server](https://github.com/petems/unsplash-mcp-server) is a Python MCP server for searching Unsplash's library of high-quality images. It's a fork of [hellokaton/unsplash-mcp-server](https://github.com/hellokaton/unsplash-mcp-server) with some tweaks.

It supports filtering by keyword, color, orientation, and pagination — so you can wire it up to an AI agent and have it find relevant stock photos as part of a larger workflow. Useful for content creation pipelines where you want an LLM to pick appropriate imagery.
